# ✅ Step 1 — Prepare Your LMS Project Structure

Your project should look like this:

```
lms-project
   client   (React / Vite / Next etc)
   server   (Node / Express backend)
```

Very important:

✅ In **server folder** you must have:

```
package.json
index.js / server.js / app.js
```

✅ In **client folder** you must have:

```
package.json
vite.config.js / next.config.js
```

**New Repository**

### 👉 Fill details

```
Repository name → lms-project
Public / Private → your wish
Do NOT initialize with README
```

✅ **Create repository**

# ✅ Step 3 — Upload Your LMS Project to GitHub

Now open **VS Code terminal** inside your LMS root folder.

Example path:

```
E:\lms-project
```

Run:

```bash
git init
git add .
git commit -m "initial commit lms"
```


---

# ✅ Step 4 — Important Change in Server (VERY IMPORTANT)

Vercel requires **serverless format**

So create file inside **server folder**

```
server/vercel.json
```

Add:

```json
{
  "version": 2,
  "builds": [
    {
      "src": "index.js",
      "use": "@vercel/node"
    }
  ],
  "routes": [
    {
      "src": "/(.*)",
      "dest": "index.js"
    }
  ]
}
```

(If your file name is `server.js` change accordingly)

---

# ✅ Step 5 — Deploy CLIENT in Vercel

### 👉 Open

[https://vercel.com](https://vercel.com)

Login using **GitHub**

### 👉 Click

**Add New → Project**

### 👉 Select

`lms-project`

Now configure:

```
Root Directory → client
Framework → auto detect
```

Click
✅ **Deploy**

After few seconds you will get:

```
https://lms-client.vercel.app
```

---

# ✅ Step 6 — Deploy SERVER in Vercel

Again click
**Add New → Project**

Select same repo again.

Now configure:

```
Root Directory → server
Framework → Other
```

Click
✅ **Deploy**

Now you will get:

```
https://lms-server.vercel.app
```

---

# ✅ Step 7 — Connect Client with Server

In your React client change API URL:

Example:

```js
axios.defaults.baseURL =
"https://lms-server.vercel.app"
```

Then again:

```bash
git add .
git commit -m "api updated"
git push
```

👉 Vercel will auto redeploy.

---

# ✅ Step 8 — Done 🎉

Now your LMS is live:

```
Frontend → vercel URL
Backend → vercel URL
Database → MongoDB / MySQL cloud
```

